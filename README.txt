IMPORTANT:
Do npm install before running project

Libraries used:
Angular: 
For Bootstrap:
npm install bootstrap@4.5.0 --save
npm install @angular/localize@11.2.6 --save
npm install @ng-bootstrap/ng-bootstrap@9.0.0 --save

For styles in angular.json:
"node_modules/bootstrap/dist/css/bootstrap.min.css"

For Express:
npm install express body-parser --save

For Axios:
npm install axios --save

For MongoDB:
npm install mongodb --save

For bcrypt.js:
npm install bcryptjs --save

For JWT:
npm install bcryptjs
npm install jsonwebtoken

For selenium:
npm install selenium-webdriver@3.6.0 
npm install chromedriver --chromedriver_version=LATEST


Anaconda - automation: 
conda install pandas
conda install matplotlib
conda install scikit-learn
conda install scipy
pip3 install pyspellchecker
pip3 install textblob
pip3 install pymongo[srv]
